Build-Android-Apps-with-JS

```
1. python generate.py -n [project_name]
2. the entrypoint for the application is in [project_name]/index.html 
3. use @include("./[PATH_TO_CSS_OR_JS]") to add js or css files to the project
4. connect your android phone
5. run run.bat or ./run.sh (if not using windows) to install the application to your phone
```
